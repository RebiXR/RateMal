## Project Context:
This project is a collaborative drawing application with special focus to accommodate users with physical or mental disabilities. 
This Project uses typeScript both as frontend and backend language. 
The backend runs on node.js and uses the Socket.io framework to establish the connection between users.
The frontend uses the React Framework.
A setup is in place for a secondary backend python fastAPI server.

## Project Structure:
- All backend related files are in the RateMal/backend folder.
- All frontend related files are in the RateMal/frontend folder.

## Code Standards:
- Use TypeScript strict mode
- Use type inference where possible
- Be explicit with types for public APIs and function parameters
